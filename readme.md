## Express Rest API

This is an example project of Rest API built with express

```bash
# rename .env.example to .env
> cp .env.example .env
# install dependencies
> npm install
# start server
> npm run dev
```

Other commands available are:

``` bash
# build for production with minification
> npm run build
> npm run start

# run unit tests
> npm run test
```

Setup with docker:

``` bash
# rename .env.example to .env
> cp .env.example .env

# start container
> docker compose up -d
```
